# Python仓库管理系统 python93

本项目是基于Python技术的仓库管理系统，整合了Django框架、MySQL数据库和Vue.js前端技术，致力于提供一套功能全面、稳定可靠的管理解决方案。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

- 商品管理
  - 商品信息维护
  - 商品分类管理
- 分类管理
  - 分类信息维护
- 用户管理
  - 用户信息维护
  - 权限控制
- 日志管理
  - 系统操作日志
- 系统信息
  - 系统状态监控
- 批量操作
  - 批量删除
- 分页查询

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- Django环境
- MySQL数据库
- Vue.js环境

## 部署步骤

1. 环境配置
   - 安装Python
   - 安装Django
   - 安装MySQL，并建立数据库
   - 安装Vue.js
2. 克隆项目代码
3. 数据库迁移和模型同步
4. 前端资源编译
5. 启动后端服务
6. 启动前端服务
